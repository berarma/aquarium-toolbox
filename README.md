# AquariumToolbox

This library can do typical calculations used in planted aquariums, like
fertilizer dosing, many others will be added.

##Requirements

  + PHP 5.4+

##How to use it

Add to your project with Composer:

```
# composer.phar require berarma/AquariumToolbox
```
